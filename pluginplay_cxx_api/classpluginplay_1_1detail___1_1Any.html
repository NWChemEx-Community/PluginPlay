Detailed Description

The Any class is capable of holding an instance of any type in a type-safe manner.

This class is used extensively through the pluginplay to avoid needing to know the types of inputs and results to a property type or module API. Under the hood it works by wrapping an std::any and casting that any to implement the additional functions which are part of the Any's API, but not the std::any's API.

◆ emplace()

template<typename T , typename... Args>
T & pluginplay::detail_::Any::emplace ( Args &&...  args)

Initializes the wrapped value by forwarding the provided arguments to the wrapped value's constructor.

This function can be used to change the instance wrapped inside this object. More specifically the arguments provided to this function are forwarded to T's constructor, and the resulting instance becomes the wrapped state (freeing up any existing state). Note the arguments can be a T instance in which case the copy or move constructor of T will be invoked.

This function is admittedly a bit intimidating until you see it in action. Let's say you have an Any instance and you want it to wrap a double, this could be done like:

Any my_any;
double& wrapped_value = my_any.emplace<double>(3.14);

Any my_any;

auto& wrapped_ve1 = my_any.emplace<std::vector<double>>(vec1);
// ...makes vec1, inside the Any, without the copy
auto& wrapped_v2 = my_any.emplace<std::vector<double>>({1.1, 1.2, 1.3});

Releases the wrapped value freeing up the memory associated with it.

Any instances own the memory they are wrapping. This member function can be used to release the memory before the Any instance goes out of scope. Following normal RAII practices the Any will release any held memory in its dtor meaning users need not call this function before letting an Any instance go out of scope.
