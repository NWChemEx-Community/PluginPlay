Detailed Description

template<typename T, AnyTag Tag = AnyTag::General>
class pluginplay::detail_::AnyWrapper< T, Tag >

Forward declare class that will implement the API.

The class responsible for holding the type-erased instance.

The AnyWrapper class holds the wrapped instance for the AnyWrapperBase class. It also is responsible for implementing all of the abstract methods

Template Parameters
TThe type of the instance to wrap. Must be copyable, hashable, and have operator== defined.
