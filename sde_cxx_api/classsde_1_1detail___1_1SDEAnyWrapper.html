Detailed Description

template<typename T>
class sde::detail_::SDEAnyWrapper< T >

Forward declare class that will implement the API.

The class responsible for holding the type-erased instance.

The SDEAnyWrapper class holds the wrapped instance for the SDEAnyWrapperBase class. It also is responsible for implementing all of the abstract methods

Template Parameters
TThe type of the instance to wrap. Must be copyable, hashable, and have operator== defined.
